Projects

Automated Toxic and Spam Detection in YouTube Comments using BERT

Comment Crafter is a web application designed to enhance content moderation by detecting and filtering toxic and spam comments on YouTube. It leverages BERT-based NLP, Rapid Fuzzy Matching, and Levenshtein Distance to analyze and classify comments efficiently. The system integrates OAuth 2.0 for secure API access and ensures improved online discourse.

Social Proximity and Mask Monitoring System

This AI-driven system ensures compliance with COVID-19 safety measures by monitoring mask adherence and social distancing in public spaces. Using CNN and YOLOv3, it detects individuals without masks and estimates crowd proximity in real-time, providing an automated solution for public safety monitoring.

Fraud Detection in E-Commerce Transactions

A machine learning-based fraud detection system that identifies suspicious credit card transactions, helping prevent financial fraud. Utilizing Logistic Regression and Decision Trees, it analyzes transaction patterns and flags potential fraud, enhancing security in online payments.

Airline Reservation and Travel Planner

A web-based flight booking system designed for a seamless travel experience. It integrates safety guides, budget-friendly travel recommendations, and real-time flight information, ensuring users find the best travel options with ease.

Virtual Mouse

A gesture-based virtual mouse that eliminates the need for physical input devices. Using computer vision and hand-tracking algorithms, it enables users to control cursor movements, clicks, and gestures through real-time hand recognition, providing a contactless and accessible computing experience.

Health Insurance Cross-Sell Prediction

A machine learning model that predicts whether existing insurance customers are likely to purchase additional health insurance. Using classification algorithms, it analyzes customer demographics, previous policy details, and behavioral data to enhance targeted marketing strategies and improve conversion rates.
